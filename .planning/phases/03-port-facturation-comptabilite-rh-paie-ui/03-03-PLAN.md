**Save mutation — double-entry insert:**
```typescript
mutationFn: async (payload) => {
  const amount = parseFloat(payload.amount)
  if (isNaN(amount) || amount <= 0) throw new Error('Montant invalide')

  // Both debit and credit must be provided
  if (!payload.debit_account_id || !payload.credit_account_id)
    throw new Error('Sélectionnez les deux comptes')

  // Balance is always valid in simplified mode (same amount debit and credit)
  const { data: entry, error } = await supabase.from('journal_entries').insert({
    date: payload.date,
    description: payload.description,
    invoice_id: payload.invoice_id || null,
    owner_id: (await supabase.auth.getUser()).data.user!.id,
  }).select('id').single()
  if (error) throw error

  await supabase.from('journal_lines').insert([
    { journal_entry_id: entry.id, account_id: payload.debit_account_id,  debit: amount, credit: 0, description: payload.description },
    { journal_entry_id: entry.id, account_id: payload.credit_account_id, debit: 0, credit: amount, description: payload.description },
  ])
},
onSuccess: () => { qc.invalidateQueries({ queryKey: ['journal'] }); closeModal(); toast.success('Écriture enregistrée') },
onError: (e: Error) => toast.error(e.message),
```

**Client-side aggregation:**
Group journal_lines by `accounts.type` and sum debits and credits:
```typescript
const totals = { asset: { debit: 0, credit: 0 }, liability: { debit: 0, credit: 0 },
  equity: { debit: 0, credit: 0 }, income: { debit: 0, credit: 0 }, expense: { debit: 0, credit: 0 } }

lines.forEach(line => {
  const type = (line.accounts as any)?.type
  if (type && totals[type as keyof typeof totals]) {
    totals[type as keyof typeof totals].debit  += Number(line.debit  ?? 0)
    totals[type as keyof typeof totals].credit += Number(line.credit ?? 0)
  }
})
```

**Income Statement (Compte de résultat):**
- Total income = `totals.income.credit - totals.income.debit` (net credits on income accounts)
- Total charges = `totals.expense.debit - totals.expense.credit` (net debits on expense accounts)
- Net result = Total income - Total charges
- Display as two-column card: left=Produits (green if positive), right=Charges (red)
- Net result row below: positive=bénéfice (green), negative=perte (red)

**Balance Sheet (Bilan):**
- Total assets = `totals.asset.debit - totals.asset.credit`
- Total liabilities = `totals.liability.credit - totals.liability.debit`
- Total equity = `totals.equity.credit - totals.equity.debit`
- Passif total = liabilities + equity
- Display as two-column layout: Actif column | Passif column
- Add note: "Bilan équilibré" badge (green) if assets ≈ passif total (tolerance 0.01), else warning badge

**Running balance computation (client-side):**
After fetching lines, compute cumulative running balance. For each line:
- `running_balance += (debit - credit)`
- Display as a fourth column. Start from 0.

```typescript
const lines = (linesData?.data ?? []).map((line, i, arr) => {
  const cumulative = arr.slice(0, i + 1).reduce(
    (sum, l) => sum + Number(l.debit ?? 0) - Number(l.credit ?? 0),
    0
  )
  return { ...line, running_balance: cumulative }
})
```
