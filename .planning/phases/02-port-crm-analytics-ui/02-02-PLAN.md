Completely replace the existing server-component contacts page with a "use client" version. This is a direct port of `frontend/src/pages/crm/Contacts.jsx` with axios replaced by Supabase client calls.

Key implementation details:

1. **Top of file:** `'use client'`

2. **Imports:**
   - useState from 'react'
   - useQuery, useMutation, useQueryClient from '@tanstack/react-query'
   - createClient from '@/lib/supabase/client'  (NOT server — this is client component)
   - toast from 'sonner'
   - Users, Trash2, Search from 'lucide-react'
   - PageShell, TableHead, EmptyRow, TableRow from '@/components/ui/PageShell'
   - Modal from '@/components/ui/Modal'

3. **Module-level singleton:** `const supabase = createClient()` — outside component to avoid re-creation on render.

4. **State:**
   - `const [search, setSearch] = useState('')`
   - `const [modal, setModal] = useState(false)`
   - `const [form, setForm] = useState({ first_name: '', last_name: '', email: '', phone: '' })`

5. **useQuery for contacts list:**
   ```typescript
   queryKey: ['contacts', search]
   queryFn: async () => {
     let q = supabase
       .from('contacts')
       .select('id, first_name, last_name, email, phone, company_id, companies(name)')
       .is('deleted_at', null)
       .order('created_at', { ascending: false })
       .limit(50)
     if (search.trim()) {
       q = q.textSearch('search_vector', search, { type: 'websearch', config: 'simple' })
     }
     const { data, error } = await q
     if (error) throw error
     return data ?? []
   }
   ```

6. **createMut (useMutation):**
   ```typescript
   mutationFn: async (f: { first_name: string; last_name: string; email: string; phone: string }) => {
     const { data: { user } } = await supabase.auth.getUser()
     const { error } = await supabase.from('contacts').insert({ ...f, owner_id: user!.id })
     if (error) throw error
   }
   onSuccess: () => { qc.invalidateQueries({ queryKey: ['contacts'] }); toast.success('Contact créé'); setModal(false); setForm({ first_name: '', last_name: '', email: '', phone: '' }) }
   onError: (e) => toast.error(e.message)
   ```

7. **deleteMut (useMutation):**
   ```typescript
   mutationFn: async (id: string) => {
     const { error } = await supabase.from('contacts').update({ deleted_at: new Date().toISOString() }).eq('id', id)
     if (error) throw error
   }
   onSuccess: () => { qc.invalidateQueries({ queryKey: ['contacts'] }); toast.success('Contact supprimé') }
   onError: (e) => toast.error(e.message)
   ```

8. **Toolbar (search input):** controlled input with Search icon, onChange sets search state. Use same styling as existing scaffold (pl-9 input with search SVG icon, border border-gray-200 rounded-lg).

9. **PageShell props:**
   - title="Contacts"
   - subtitle=`${contacts?.length ?? 0} contact(s)` (handle isLoading: show '…' if loading)
   - icon={Users}, iconBg="bg-violet-100", iconColor="text-violet-600"
   - actionLabel="Nouveau contact"
   - onAction={() => setModal(true)}
   - toolbar={searchInput JSX}

10. **Table columns:** Nom (first_name + last_name), Email, Téléphone, Entreprise (companies?.name or '—'), Actions
    - Last column: Trash2 icon button, onClick={() => deleteMut.mutate(c.id)}, className="p-1.5 rounded hover:bg-red-50 text-gray-400 hover:text-red-500 transition"

11. **isLoading state:** show `<EmptyRow colSpan={5} icon={Users} message="Chargement…" />` while loading.

12. **Modal create form:**
    - `<Modal open={modal} onClose={() => setModal(false)} title="Nouveau contact" size="md">`
    - Form fields: Prénom (first_name required), Nom (last_name required), Email (type="email"), Téléphone
    - Input classes: `w-full border border-gray-200 rounded-lg px-3 py-2 text-sm focus:outline-none focus:ring-2 focus:ring-violet-500`
    - Submit button: bg-violet-600 text-white, disabled={createMut.isPending}, label "Création…" when pending
    - Cancel button: border border-gray-200 text-gray-700 hover:bg-gray-50

13. **TypeScript:** type the companies embed as `{ companies: { name: string } | null }` joined with the contacts Row type from database.types.ts. Use `Database['public']['Tables']['contacts']['Row']` for base type.

IMPORTANT constraints:
- NEVER use .delete() — use .update({ deleted_at: ... }) for soft delete (STATE.md Decision #6)
- NEVER use lib/supabase/server in this file — it's "use client"
- ALWAYS filter .is('deleted_at', null) in queries
